# Employee Attendance & Leave Management System

A simple Java Spring Boot + MySQL backend application for managing employee attendance and leave requests.

## Features
- Punch IN/OUT for attendance  
- Apply for leave  
- View pending leave requests  

## Tech Stack
- Java  
- Spring Boot  
- MySQL
